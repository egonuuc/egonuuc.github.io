Trump-Libs Class Project

Tools/Languages Used: HTML, CSS, Bootstrap, Git, JavaScript, MongoDB, Various APIs

The Donald Trump Tweet Generator project was a part of my web programming class. It was a group project in which we used tried to use different tools to create a game. In our case, we decided to make a tweet generator that functioned like the game mad-libs. Users would input three words and it would re-populate an already existing Donald Trump tweet with their words. For this project, we used HTML, CSS, and JavaScript for the front-end. Additionally, I created a background image for one of the pages on Photoshop. For creating the tweets, we stored Donald Trump's tweets in a database using MongoDB. We also used the Merriam-Webster Dictionary api to determine the parts of speech of the user input. Finally, we used Git extensively to share work as a team for this project.

NotUber Class Homework

Tools/Languages Used: HTML, CSS, JavaScript, MongoDB, Heroku, Gooogle Maps API

NotUber was an assignment for my Web Programming class. It was broken into two parts: the front-end aspect and the back-end aspect. For the front-end, we learned how to use JavaScript by creating the interface for a hypothetical ride-sharing app. I learned how to use functions from the Google Maps API and wrote JavaScript functions to display a map of my location and calculate distance from different points. In addition to this, we implemented the back-end for the app suing MongoDb. In this, I created a database of both passengers and drivers and created functions to show which drivers were close by to a passenger at a certain point. I also learned how to deploy apps to Heroku sing this process.
